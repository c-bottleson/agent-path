Contabo

  6. Operating System > Ubuntu 24.04 (or 22.04)
  7. Password: Have them generate and save a root password.

Vultr
  1. Go to vultr.com and Sign Up. Add CC.
  2. Go to Deploy New Instance
  3. Choose Cloud Compute > Shared CPU
  4. Location: Closest to them.
  5. OS Image > Ubuntu 24.04 LTS
  6. Server Size: 1 vCPU / 2GB RAM ($10/mo).
  7. Uncheck "Enable automated backups" (save $2).
  8. Click Deploy Now. IP available immediately.

🔴 Must-Cover Concepts (Before Ending Call 1)

These three concepts from the core curriculum must be planted during the first call, even briefly. They're the foundation for everything in Calls 2-4.

1. "Hope is Not a Strategy" (The Watchdog Rule)

"Every automation you build, we also build something that watches it. If your scraper breaks at 2 AM, the agent texts you at 2:01 AM—not when you check two weeks later. We'll set this up properly in our next call."

2. State Injection (Don't Bloat the Brain)

"Don't dump everything into the agent's memory at once. Instead, we create little instruction files for each project. When you say 'work on the CRM,' it reads only the CRM file. Keeps it fast and cheap."

3. The "Probably Not An Issue" Trap

"If the agent ever says an error is 'probably fine'—push back. Make it prove it. Hope is not a strategy. If it can't prove it, we build a watchdog to catch it."



The Autonomy Roadmap (Follow-up Calls)

"This first call just puts the employee at the desk. The next three calls actually teach them how to do your job so you can step away."

Call 2: Memory & Skills

"Teaching them the ropes"

We configure persistent memory and Standard Operating Procedures (Skills) so you stop repeating yourself.

If Hermes (The COO):
  • Teach exact formatting for inbound CRM leads.
  • Provide read-only API access to Stripe/DBs.
  • Lock in Telegram communication styles.
If OpenClaw (The Hacker):
  • Teach codebase architecture patterns.
  • Build a custom "Web Scraping" module for their niche.

Call 3: Watchdogs & Crons

"Working while you sleep"

We remove the need to "click run". The system wakes up on its own to do the work autonomously.

If Hermes (The COO):
  • Watchdogs: Silent triggers that text you only if servers drop or prices shift.
  • Crons: 6:00 AM daily briefings pulled directly from your databases.
If OpenClaw (The Hacker):
  • Automated Execution: Cron jobs that wake, scrape/trade, and go back to sleep.

Call 4: The Idea Pipeline

"Human as CEO"

Full autonomy. You stop managing tasks and only provide ideas. The system orchestrates itself.

If Hermes (The COO):
  • Idea Ingestion: Forward an email or voice memo. Hermes parses it, plans it, and sub-delegates it.
If OpenClaw (The Hacker):
  • End-to-End: Drop a Jira ticket in chat. It writes the code, runs tests, commits, and deploys.
