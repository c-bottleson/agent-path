PHASE 1: DECISIONS

Three quick choices. Takes about 5 minutes.

1 Agent Engine

"We're going to install an AI agent on your server. Think of it as a digital employee that works 24/7. There are two engines — the difference is what they're bad at."

"Hermes is your operations manager. Great at managing systems, sending you messages, running background tasks, and talking to APIs. Its weakness: it's not a data scientist. It won't train ML models or do heavy number-crunching."

"OpenClaw is your data scientist. Built for Python, machine learning, and complex computation. Its weakness: it's heavier, uses more RAM, and isn't as snappy for day-to-day admin tasks."

"For most people, Hermes is the right call. It's lighter, faster, and does 90% of what you'd want. OpenClaw is the choice if you specifically need data science or ML."

🖱️ Select Agent

2 VPS (Server)

"Your agent needs a home — a server that runs 24/7. Two options:"

"Contabo is the work truck. ~$6/month, massive hardware for the price. The website looks like it's from 2005, but the server is a beast. If you just want raw power and don't care about fancy dashboards, this is it."

"Vultr is the sports car. ~$10/month. Sleek dashboard, instant provisioning, NVMe drives. If you think you'll log into the dashboard often, this is nicer to use."

"Both work perfectly. Contabo is more bang for buck. Vultr is more polished."

🖱️ Select VPS

3 LLM Brain

"The brain is the AI model that powers your agent. We access all models through OpenRouter — one API key, every model. Two tiers:"

"Top tier: Claude Opus 4.7 or GPT 5.5. These are the smartest models on the planet right to. Flawless code, complex reasoning, nuanced decisions. Roughly $15-30/month with typical usage."

"Mid tier: Claude Sonnet 4.6 or Gemini 2.5 Pro. Excellent at almost everything — code, writing, analysis. Not quite as sharp as Opus/GPT-5.5 on the hardest problems, but honestly? For 95% of tasks you won't notice a difference. Roughly $5-15/month."

"My recommendation: start mid tier. It's genuinely great. Your agent can upgrade to top tier later for specific heavy tasks — it's smart enough to pick the right model for the job. But if you want the absolute best from day one, go top tier."

🖱️ Select Model Tier

💡 Note: Your agent can switch between models at any time. This just sets the default. As it learns your needs, it'll pick the right model for each task automatically.

PHASE 2: ACCOUNTS & CREDENTIALS

Customer follows along on their machine. Save everything to notepad. ~15 minutes.

🖥️ Sidequest A: Create the VPS

⏱ Note: VPS provisioning takes 15-30 minutes on Contabo (instant on Vultr). We'll set up the other accounts while we wait.

Have them share their screen. Walk through this together:

  1. Go to contabo.com and click Cloud VPS
  2. Select Cloud VPS 1 ($5.99/mo) — this is plenty for an AI agent
  3. Term Length: 1 Month (no long commitment)
  4. Region: Pick the closest to them (or cheapest)
  5. Storage: 50GB NVMe
  6. ⚠️ Operating System: Ubuntu 24.04 LTS — this is critical, do NOT pick Windows or Debian
  7. Password: Click "Generate" and have them save it in their notepad as:
    VPS Root Password: _______________
  8. Check out with credit card
  9. Save the server IP when it arrives (15-30 min email):
    VPS IP Address: _______________

Have them share their screen. Walk through this together:

  1. Go to vultr.com and sign up. Add a credit card.
  2. Click Deploy New Instance
  3. Choose Cloud Compute → Shared CPU
  4. Location: Closest to them
  5. ⚠️ OS Image: Ubuntu 24.04 LTS — do NOT pick anything else
  6. Server Size: 1 vCPU / 2GB RAM ($10/mo)
  7. Uncheck "Enable automated backups" (saves $2/mo)
  8. Click Deploy Now — IP is available immediately
  9. Save in notepad:
    VPS IP Address: _______________
    VPS Root Password: _______________
    (password shown on the instance details page)

💬 Sidequest B: Create a Telegram Bot

"Your agent talks to you through Telegram — like texting a friend. We need to create a bot for it. This takes about 2 minutes."
  1. Open Telegram on their phone (or web.telegram.org on desktop)
  2. Search for @BotFather and start a chat
  3. Type /newbot and send it
  4. BotFather asks for a name: suggest something like "My AI Assistant" (this is the display name)
  5. BotFather asks for a username: must end in bot. Suggest "theirname_agent_bot"
  6. BotFather replies with a token — looks like 123456789:ABCdefGHIjklMNOpqrsTUVwxyz
  7. Save in notepad:
    Telegram Bot Token: _______________
⚠️ Tell them: "This token is like a password for your bot. Don't share it publicly."

Now get their Telegram User ID:

"One more thing — I need your Telegram user ID so the bot knows to talk to you and not strangers. This takes 10 seconds."
  1. In Telegram, search for @userinfobot
  2. Start a chat and send /start
  3. It replies with a numeric ID (something like 8164757328)
  4. Save in notepad:
    Telegram User ID: _______________
💡 Explain: "This ID is how your bot knows it's you. Without it, anyone could talk to your agent."

🧠 Sidequest C: Get the AI Brain (OpenRouter)

"Now we need to connect the brain. OpenRouter is like a universal adapter — one account, every AI model. You're going to add $10 in credits to start."
  1. Go to openrouter.ai and sign up (Google/GitHub login works)
  2. Click the Wallet/Credits icon in the top right
  3. Click Add Credits → add $10 via credit card
  4. Set up auto-refill (important — keeps the agent running):
    • On the same Wallet page, look for "Auto Refill" or "Recurring"
    • Enable it: $10 when balance drops below $2
    • This way the agent never runs out of brainpower
  5. Go to Keys in the top menu bar
  6. Click Create Key → name it "VPS Agent"
  7. Copy the key (starts with sk-or-v1-...)
  8. Save in notepad:
    OpenRouter API Key: _______________
💡 Explain: "With auto-refill, you'll spend roughly $5-15/month on the AI brain. That's it. No subscription, no markup — you're paying the AI provider directly."

📂 Sidequest D: Set Up GitHub

"GitHub is where your agent stores code and projects. Think of it as Google Drive, but for software. Even if you've never used it, your agent needs it to build things for you."
  1. Go to github.com and sign up (free account is fine)
  2. Choose the Free plan
  3. Verify their email
  4. Create a Personal Access Token (this lets the agent manage repos):
    • Click profile picture (top right) → Settings
    • Scroll down to Developer settings (left sidebar)
    • Personal access tokens → Tokens (classic)
    • Click Generate new token → Generate new token (classic)
    • Note: VPS Agent
    • Expiration: 90 days (renew later)
    • ⚠️ Scopes — check these: repo, workflow, read:org
    • Click Generate token
  5. Save in notepad:
    GitHub Username: _______________
    GitHub Token: _______________
💡 Why this matters: "With this token, your agent can create repos, push code, and manage projects on your behalf. When it builds something, it saves it to GitHub automatically."

🌐 Sidequest E: Set Up Vercel

"Vercel is where your agent deploys websites and web apps. When your agent builds something, this is where it goes live. Free tier is plenty to start."
  1. Go to vercel.com and click Sign Up
  2. Sign in with GitHub (use the account they just created) — this auto-connects the two
  3. Accept the free Hobby plan
  4. That's it — no tokens needed. Vercel deploys automatically when code is pushed to GitHub.
  5. Save in notepad:
    Vercel Account: connected to GitHub ✓
💡 How it works: "When your agent builds a website, it pushes the code to GitHub, which auto-deploys to Vercel. You get a live URL instantly. Free tier: 100GB bandwidth."

REFERENCE: Concepts to Plant (Before Ending Call)

Teach these briefly. Seeds for Calls 2-4.

1. "Hope is Not a Strategy" (The Watchdog Rule)

"Every automation you build, we also build something that watches it. If your scraper breaks at 2 AM, the agent texts you at 2:01 AM — not when you check two weeks later."

2. Don't Bloat the Brain

"Don't dump everything into the agent at once. Create little instruction files for each project. When you say 'work on the CRM,' it reads only the CRM file. Keeps it fast and cheap."

3. The "Probably Fine" Trap

"If the agent says an error is 'probably fine' — push back. Make it prove it. If it can't, we build a watchdog to catch it."

The Autonomy Roadmap (Follow-up Calls)

"This first call puts the employee at the desk. The next three teach them to do your job."

Call 2: Memory & Skills

"Teaching them the ropes"

Persistent memory and SOPs so you stop repeating yourself.

  • Teach exact formatting for your workflows
  • Connect to your CRM, Stripe, databases
  • Build custom skills for your niche

Call 3: Watchdogs & Crons

"Working while you sleep"

Remove the need to "click run."

  • Watchdogs: Silent triggers that text you only if something breaks
  • Crons: Daily briefings, automated reports

Call 4: The Idea Pipeline

"Human as CEO"

Full autonomy. You provide ideas, it executes.

  • Forward an email or voice memo — agent parses, plans, and builds
  • End-to-end project execution
